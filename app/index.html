Sojourn

Highly disposable front-end boilerplate.

When you don't want to spend all day getting ready to start

The next time you launch a front-end heavy project, do yourself a favor and get the minor decisions out of the way. Think about your brilliant idea, not the tools you're going to use to deliver it.

Deliberately not a framework, Sojourn is just a curated, preconfigured assembly of pieces into a healthy toolchain. Clone it, zap its git history, start a new project that's all yours and hit the ground running!

Script
Sojourn is bundled with a toolchain that embraces testing and modular code composition. When you're done, "grunt build" to compile everything into a single exportable file (minified version included).
Visualize
Embark on a wondrous prototyping experience without needing to take the time to install any number of visual tools. Sojourn comes pre-bundled with Sass, Bourbon + Neat, and Font Awesome. Add any new imagery or fonts with ease in the app/images and app/fonts directories.

It's a la carte. Because of the way the build system is laid out, you can use or not use anything except Grunt itself. It still gets collated and made ready for production. Don't like CoffeeScript? I guess we can still be friends. Your Javascript will compile just fine. Same with Backbone, React, jQuery, etc. Don't require it, and it just won't be there.

Get started now
  • git clone git@github.com:esmevane/sojourn
  • cd sojourn
  • npm install
  • rm -rf .git
  • npm install -g grunt-cli
  • grunt serve

Sojourn will observe changes to any CoffeeScript or JavaScript file in assets/scripts/**, or any Sass file in assets/styles/**. When any of those files change, it will bootstrap a new copy of the application files and a simple refresh will show you your work.
